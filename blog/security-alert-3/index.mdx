A potential consensus issue with Parity Ethereum has been identified.

**Severity:** Critical  
**Product affected:** Parity Ethereum client  
**Affected versions:**

* stable track up to 1.10.5-stable
* beta track: up to 1.11.2-beta

**Mitigation:**

The issue has been fixed in the newly released Parity Ethereum client versions:

* [1.10.6-stable](https://github.com/paritytech/parity/releases/tag/v1.10.6)
* [1.11.3-beta](https://github.com/paritytech/parity/releases/tag/v1.11.3)

Please update your nodes as soon as possible and then double check that you are running version 1.10.6-stable or 1.11.3-beta.
